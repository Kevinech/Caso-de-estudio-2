# Biblioteca Digital Estudiantil

Aplicación sencilla en Windows Forms para registrar y buscar libros, creada con fines académicos para practicar algoritmos de búsqueda.

## Uso rápido

1. Descarga el repositorio completo.
2. Abre la solución en Visual Studio.
3. Ejecuta el proyecto y comienza a usar la biblioteca digital.

## Características

- Registro de libros: Nombre, Autor, Categoría, Código.
- Búsqueda por Nombre, Autor, Código y Categorías dinámicas.
- Validación: No se pueden agregar libros con campos vacíos ni códigos repetidos.
- Visualización con DataGridView para filtrar resultados.
